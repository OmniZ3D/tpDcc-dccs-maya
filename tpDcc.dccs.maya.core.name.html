class tpDcc.dccs.maya.core.name.FindUniqueName(name)¶

Bases: tpDcc.libs.python.name.FindUniqueString, object

This class allows to find a name that does not clash with other names in the Maya scene It will increment the last number in hte name If no number is found, it will append a 1 to the end of the name
