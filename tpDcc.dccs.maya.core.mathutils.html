tpDcc.dccs.maya.core.mathutils.get_axis_vector(transform, axis_vector)¶

Returns the vector matrix product If you give a vector [1, 0, 0], it will return the transform’s X point If you give a vector [0, 1, 0], it will return the transform’s Y point If you give a vector [0, 0, 1], it will return the transform’s Z point :param transform: str, name of a transforms. Its matrix will be checked :param axis_vector: list<int>, A vector, X = [1,0,0], Y = [0,1,0], Z = [0,0,1] :return: list<int>, the result of multiplying the vector by the matrix Useful to get an axis in relation to the matrix
